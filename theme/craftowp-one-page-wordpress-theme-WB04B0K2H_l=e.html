CraftoWP - One-Page WordPress Theme

  • Details
  • Comments
Live preview
  • Description
  • Changelog

Need a nice place where to showcase your work. CraftoWP is a one-page portfolio Wordpress theme which let you present yourself to the world exactly as you are even if you are professional or agency. CraftoWP comes with many built-in features like Tinymce 4 shortcodes button with more than 20 ready shortcodes, drag-and-drop elemets, Google Fonts, customizable backgrounds and colors and many more.

Even if you want to keep all content in a one-page website or you want to have additional pages, CraftoWP will help you achieve the right image for you by letting you create portfolio galleries with expandable previews and sortable categories.

Along with the 5 Minutes Setup video, the theme comes with very descriptive and complete documentation in HTML format for a better navigation and search. Also you'll get a theme with many expanded functions directly on Post/Page edit screen or Theme Options page.

About Bootstrap

Bootstrap is designed to help people of all skill levels – designer or developer, huge nerd or early beginner. Use it as a complete kit or use it to start something more complex.

Unlike other front-end toolkits, Bootstrap was designed first and foremost as a styleguide to document not only its features, but best practices and living, coded examples.

Built to support new HTML5 elements and syntax with progressively enhanced components.
