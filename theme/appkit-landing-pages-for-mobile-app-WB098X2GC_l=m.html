About

AppKit is an HTML template for creating a landing page for your mobile app. The pack supports three major OS: iOS, Android and Windows Phone. There are various mobile devices, including tablets, provided for each OS – all of them have color options.

The main elements of AppKit are «Cover» and «Landing page». Cover is a very lightweight page and provides a user with a quick overview of your app. It also leads to to the landing page for a better introduction. You can use both pages as well as one of them separately.

The template is built on the basis of a popular library - Bootstrap 3.3.4. Along with standard scripts, AppKit uses Bower package manager for uploading and updating libraries. The scripts are divided into simple parts. You can easily activate or disable any feature. AppKit is easy to start&use and does not require graphic editors. All you need is to take screenshots of your app and insert them into the template, choosing the relevant device via classes. AppKit is provided with a large number of PSD mock-ups, developed specifically for this template. Insert a screenshot of your application into the mock-up for any device, export into image and put to the template. The process is quite simple as well.

Features area includes several interactive sections for a detailed description of your application. The template also includes a working subscription form and template pages for the company description, FAQ and terms of use.

About Bootstrap

Bootstrap is designed to help people of all skill levels – designer or developer, huge nerd or early beginner. Use it as a complete kit or use it to start something more complex.

Unlike other front-end toolkits, Bootstrap was designed first and foremost as a styleguide to document not only its features, but best practices and living, coded examples.

Built to support new HTML5 elements and syntax with progressively enhanced components.

Browser support

Bootstrap is tested and supported in major modern browsers like Chrome, Firefox, and Internet Explorer.

  • Latest Chrome
  • Latest Firefox
  • Latest Safari
  • Latest Edge
  • Internet Explorer 10/11
  • Latest Opera

Features & includes

Bootstrap provides an unparalleled array of features and reusable components that let you hit the ground running:

  • Responsive 12-column grid
  • 12 custom, modular jQuery plugins
  • CSS styles for forms, navigation & more
  • Dozens of reusable components
  • Components are scaled according to a range of resolutions and devices
  • Built with Sass (CSS included)
  • Complete styleguide documentation
