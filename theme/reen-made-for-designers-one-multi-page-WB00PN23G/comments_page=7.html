Hi!

Add the following styles in a custom.css file after the main.css and the color stylesheet or at the end of the main.css:

.post-title a:hover,
.post-content .meta a:hover { /* Replace .meta with .categories if you only want the categories without hover! */

color: #1ABB9C; /* Depends on the color stylesheet you use! */

How can I make the logo larger in size?

Jon
fuviz SELLER
2 years ago
Hi,

Thanks a lot for using REEN and for your kind words!

You need to change 2 - 3 values in the main.css at the final section SKROLLR ANIMATIONS (end of document) for resizing the logo:

.navbar .navbar-brand img {
height: 80px; /* Initial height – change to your needs */
-skrollr-animation-name: shrink-logo;
}

@-skrollr-keyframes shrink-logo {
40 {
max-height: 80px; /* Height after 40px (top bar height) scrolling – should be the same as the initial height. Change to your needs */
}

300 {
max-height: 20px; /* Height after 300px scrolling. Change to your needs */
}
}

Hope this helps!

Bootstrap is designed to help people of all skill levels – designer or developer, huge nerd or early beginner. Use it as a complete kit or use it to start something more complex.

Unlike other front-end toolkits, Bootstrap was designed first and foremost as a styleguide to document not only its features, but best practices and living, coded examples.

Built to support new HTML5 elements and syntax with progressively enhanced components.
