smnedelko SELLER

Hi sandrapuma55,

thanks for the comment.

1. What do you mean when you saying that code quality is very bad? Could you be more specific, may be bad code examples?

2. File size 1.5MB must not surprising you, because by default pixeladmin.min.js contains ALL integrated plugins (47 plugins + PA core) - that's why Builder exists. Take any template and concatenate all included plugins - resulting file size will be not much smaller(or even bigger). Just exclude all unused plugins + configure Gzip compression for assets on the server and file size will be much smaller. If you don’t want to use the concatenated version, you can include js files separately from the requirejs assets (it will work even if you don't use RequireJS).

3. I don't find the structure complicated - just use assets from the dist directory, which contains compiled css and js(concatenated and requirejs versions) files. Source directories, such as js or scss, contain .scss and .js files that are converted by sass and babel on compilation. Anyway, improved package directory structure and build process will be available on the upcoming update.

4. Some core components, such as px-nav/px-navbar/px-sidebar/px-footer, are deeply integrated with PA. You can use it outside PA, but it will require from you spend some time for integration.

5. What do you mean by customization?
- you can customize theme (for example height of navbar, width of px-nav, height of px-nav and px-nav dropdown items, toastr icons, dropzone icon size and much more) just by editing scss variables;
- you can add custom colors with one line of code;
- you can create themes using scss mixins;
- you can exclude unused components...

P.S. I have been programming for more than 10 years and I also was a backend developer, so I know how it work.

Anyways, if you have integration/customization questions, you always can email me.

Best regards, Sergei.

About Bootstrap

Bootstrap is designed to help people of all skill levels – designer or developer, huge nerd or early beginner. Use it as a complete kit or use it to start something more complex.

Unlike other front-end toolkits, Bootstrap was designed first and foremost as a styleguide to document not only its features, but best practices and living, coded examples.

Built to support new HTML5 elements and syntax with progressively enhanced components.

Browser support

Bootstrap is tested and supported in major modern browsers like Chrome, Firefox, and Internet Explorer.

  • Latest Chrome

  • Latest Safari
  • Latest Edge
  • Internet Explorer 10/11
  • Latest Opera

Features & includes

Bootstrap provides an unparalleled array of features and reusable components that let you hit the ground running:

  • Responsive 12-column grid
  • 12 custom, modular jQuery plugins
  • CSS styles for forms, navigation & more
  • Dozens of reusable components
  • Components are scaled according to a range of resolutions and devices
  • Built with Sass (CSS included)
