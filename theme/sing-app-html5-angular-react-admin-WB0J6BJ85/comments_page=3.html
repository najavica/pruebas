Purchased the theme and had a chance to play with it today. It's the most beautiful theme out there IMO. My biggest gripe is that the size of the core is to big even without the extras. The build/deployment could also be easier.

1. Maybe consider https://github.com/bramstein/fontfaceobserver?

2. I couldn't do proper offline development due to the Google fonts dependency. Maybe I am missing something since I see the fonts folder does have the Google fonts installed but the font declarations don't make use of it. There are quite a few approaches to fall back to local fonts. Fonfaceobserver should mitigate this in any case.

3. In your vars.scss the font is being referenced via ../vendor folder which means the expectation is that the entire vendor folder be published to the public directory in production. This isn't normal (for example, see the dependencies you use). The correct approach is to remove those overrides from var, use the default approach and instead just copy fonts to the dist/public/simialr folder. I personally prefer copying fonts with gulp-flatten.

4. For production, it would be nice if you published the compiled assets directly such as sing-core.js and sing-extras.js for easier deployment. In either case, I had to reference your preview footer and create it myself using gulp-includer to make management easier (your documentation regarding bower is obsolete so wasn't much help).

5. The documentation around bower is since it mentions removing stuff after line 23 in ajax/ folder. Either way #4 will make this easier since the dependencies will be explicit in sing-core.js, etc.

About Bootstrap

Bootstrap is designed to help people of all skill levels – designer or developer, huge nerd or early beginner. Use it as a complete kit or use it to start something more complex.

Unlike other front-end toolkits, Bootstrap was designed first and foremost as a styleguide to document not only its features, but best practices and living, coded examples.

Built to support new HTML5 elements and syntax with progressively enhanced components.

Browser support

Bootstrap is tested and supported in major modern browsers like Chrome, Firefox, and Internet Explorer.

  • Latest Chrome
  • Latest Firefox
  • Latest Safari
  • Latest Edge
  • Internet Explorer 10/11
  • Latest Opera

Features & includes

Bootstrap provides an unparalleled array of features and reusable components that let you hit the ground running:

  • Responsive 12-column grid
  • 12 custom, modular jQuery plugins
  • CSS styles for forms, navigation & more
  • Dozens of reusable components
  • Components are scaled according to a range of resolutions and devices
  • Built with Sass (CSS included)
  • Complete styleguide documentation
