Is there documentation on how to upgrade from angular5 (bootstrap3) to angular5 (bootstrap4)?
themicon SELLER
2 months ago
Hi, Bootstrap 4 by itself is not backward compatible, meaning many things changed. Of course, everything starts by updating dependencies in package.json but then the markup and styles have changed a lot.
Here is a list of changes made in the framework: https://getbootstrap.com/docs/4.0/migration/

The simple way to update is to identify custom changes to the current Bootstrap 3 version and migrate them into them BS4 version, this way will be more simple than dealing with the full set of changes in the template.
