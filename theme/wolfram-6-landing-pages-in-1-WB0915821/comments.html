YevSim SELLER
1 year ago
Hi,

I do confirm this bug. It seems to be caused by position: fixed; of the background image which is disabled by iPhone by default. As a temporary solution, please open assets/css/style.css and find this part:

body.classic .welcome_classic {
display: table;
height: 100%;
min-height: 100%;
width: 100%;
background-image: url(../img/bg-pattern_alt.png), url(../img/screen-bg_11.jpg);
background-position: top left, center center;
background-size: auto, cover;
background-repeat: repeat, no-repeat;
background-attachment: fixed;
color: white;
text-shadow: 1px 1px 2px rgba(0, 0, 0, 0.18);
}

and add the following code right after it:

@media(max-width: 767px) {
body.classic .welcome_classic {
background-attachment: scroll;
}
}

This will disable the "parallax" behavior on mobile devices, and make the background image look as it is supposed to look. Please let me know if you have any other questions.
