Hi,

How can I modify ionicon svg files to make them two tone like they are in the sidebar. I've spent days with Inkscape to try and get them to work but can only manage to get them as a single inherited color. Can you explain how to make it two?

Thanks
themicon SELLER
6 months ago
Hi, you can customize color by creating a new theme.
For example, in file components/common/themes.less you can find defined themes included with the template.
Each theme is a class that used mixin to setup the colors used, in case of themes that uses two tones svg a mixin named .theme-sidebar-items is used, this mixin receives two params, the primary and second color.
In short, you can create a define new color by using the mixin like this:

// file: components/common/themes.less
.my-theme {
.theme-sidebar-items(@brand-primary, @brand-success);
}
The, you can add .my-theme class to body element in order to make it the default theme.

Note that other mixins are used in order to customize the sidebar, header, etc.
The svg files doesn't contains information about colors, the color are inherited from what is define in css.

Let us know if you have any question
Regards

Hi again, you're right, svg were modified only to be used in the sidebar, sorry we forgot to mention that in previous answer.
You need to add fill="currentColor" attribute to elements in SVGs files for elements that will contain a different color.
In the mentioned mixin, .theme-sidebar-items we used two properties, for your reference:
{

fill: @color2 !important; <-- this will apply to elements without "fill" attribute
}

Other SVGs files for Ionicons can be obtained from bower or npm packages installed on your system, or you can download theme manually from its repository:

About Bootstrap

Bootstrap is designed to help people of all skill levels – designer or developer, huge nerd or early beginner. Use it as a complete kit or use it to start something more complex.

Unlike other front-end toolkits, Bootstrap was designed first and foremost as a styleguide to document not only its features, but best practices and living, coded examples.

Built to support new HTML5 elements and syntax with progressively enhanced components.

Browser support

Bootstrap is tested and supported in major modern browsers like Chrome, Firefox, and Internet Explorer.

  • Latest Chrome
  • Latest Firefox
  • Latest Safari
  • Latest Edge
  • Internet Explorer 10/11
  • Latest Opera

Features & includes

Bootstrap provides an unparalleled array of features and reusable components that let you hit the ground running:

  • Responsive 12-column grid
  • 12 custom, modular jQuery plugins
  • CSS styles for forms, navigation & more
  • Dozens of reusable components
  • Components are scaled according to a range of resolutions and devices
  • Built with Sass (CSS included)
  • Complete styleguide documentation
