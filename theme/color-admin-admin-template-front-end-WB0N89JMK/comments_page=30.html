Nektarios
seanngu SELLER
1 month ago
Hi Nektarios,

Regarding your question, you might need to write an custom script to expand / collapse all the accordion items. Below is an simple example for how to expand / collapse all accordion items

e.g
<a href="#" data-click="expand-all">Expand All Accordions</a>
<a href="#" data-click="collapse-all">Collapse All Accordions</a>

<script>
$('[data-click="expand-all"]').click(function(e) {
e.preventDefault();
$('[data-toggle="collapse"]').closest('.card').find('.collapse').addClass('show');
});
$('[data-click="collapse-all"]').click(function(e) {
e.preventDefault();
$('[data-toggle="collapse"]').closest('.card').find('.collapse').removeClass('show');
});
</script>

Best Regards,
Sean.

Built to support new HTML5 elements and syntax with progressively enhanced components.
