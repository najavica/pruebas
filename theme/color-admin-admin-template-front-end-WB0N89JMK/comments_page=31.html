Thank you Sean. Another question: How do I hide a column? I have a table with 10 columns and I want to show only 5 columns. The other columns I want to make available when the user clicks (+). Can you help me? It is possible?

Regards,
seanngu SELLER
3 weeks ago
Hi Isandro,

You may use some data attribute to control which column you want to hide / show with the css hide.
E.g

<td class="hide" data-id="extra-col">...</td>
<td class="hide" data-id="extra-col">...</td>
<td class="hide" data-id="extra-col">...</td>
<td class="hide" data-id="extra-col">...</td>

<script>
$('[data-click="toggle-hidden-col"]').live('click', function(e) {
e.preventDefault();
if ($('[data-id="extra-col"]').hasClass('hide')) {
$('[data-id-"extra-col"]').removeClass('hide');
} else {
$('[data-id-"extra-col"]').addClass('hide');
}
});
</script>

Best Regards,
Sean.

About Bootstrap

Bootstrap is designed to help people of all skill levels – designer or developer, huge nerd or early beginner. Use it as a complete kit or use it to start something more complex.

Unlike other front-end toolkits, Bootstrap was designed first and foremost as a styleguide to document not only its features, but best practices and living, coded examples.

Built to support new HTML5 elements and syntax with progressively enhanced components.
