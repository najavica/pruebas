Where is the speed of the jumbotron transition controlled?
YevSim SELLER

Hi jalbre2,

Thanks for your question. You can easily do this with CSS. Please open the ui-elements.css file and add the following code to the very end of it:

.carousel-inner > .item {
-webkit-transition: 3s ease-in-out left;
-moz-transition: 3s ease-in-out left;
-o-transition: 3s ease-in-out left;
transition: 3s ease-in-out left;
}

Replace the 3s interval with any number you need.

Hi,

Unloaded images can throw off Isotope layouts and cause item elements to overlap. This is a known isotope issue. Please open the isotope-custom.js file and try to replace everything with:

var $container = $('#isotope-container').imagesLoaded( function() {
$container.isotope({
itemSelector : '.isotope-item',
});
});
$('#filters a').click(function(){
var selector = $(this).attr('data-filter');
$container.isotope({ filter: selector });
return false;
});

Please send me an email if this didn't work for you.

YevSim SELLER

Hi John,

I suppose you are talking about the main slideshow (i.e. carousel) on the homepage of the template. Please open index.html and find the following line:

<div id="main-slideshow" class="carousel slide" data-ride="carousel">

Then add the data-interval attribute to control "the amount of time to delay between automatically cycling an item. If false, carousel will not automatically cycle."

<div id="main-slideshow" class="carousel slide" data-ride="carousel" data-interval="5000">

5000 is the default value.

About Bootstrap

Bootstrap is designed to help people of all skill levels – designer or developer, huge nerd or early beginner. Use it as a complete kit or use it to start something more complex.

Unlike other front-end toolkits, Bootstrap was designed first and foremost as a styleguide to document not only its features, but best practices and living, coded examples.

Built to support new HTML5 elements and syntax with progressively enhanced components.
