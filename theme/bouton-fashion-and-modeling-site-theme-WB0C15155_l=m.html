In the fashion and design world, appearances are everything. Represent your brand with a chic and sophisticated website using our professional template. Introducing Bouton, a multi-purpose HTML template with excellent layout and functionality. The Bouton template features easy and effective website navigation, maximum usability, and clean content display to effectively target your audience.

With its easy-to-use interface and appealing design options, Bouton stands out as the ideal template for any modeling agency, design studio, or fashion brand looking to make a memorable impression on their clients.

Feel free to build your personal portfolio or create a new brand inspired by our design. Our template is easily adaptable, allowing you to construct a website with whichever style or layout fits your brand without affecting design or coding quality.

The Bouton template offers a new take on creative agency websites while making use of the latest web development technologies. The template is fully customizable and compatible with IE8+, Firefox, Google Chrome, Opera and Safari. The template supports Twitter Bootstrap 3.x framework and easily allows the user to edit as they go. The template design contains three home page layouts, a theme editor, and 15 page elements to choose from including footers, headers, menus, typographic elements and buttons. Detailed documentation is included to assist you in customizing to the template to your own needs with ease.

This template makes designing a website a seamless experience optimized to drive tangible results, helping you boost customer retention and gathering a loyal following with every new visit.

About Bootstrap

Bootstrap is designed to help people of all skill levels – designer or developer, huge nerd or early beginner. Use it as a complete kit or use it to start something more complex.

Unlike other front-end toolkits, Bootstrap was designed first and foremost as a styleguide to document not only its features, but best practices and living, coded examples.

Built to support new HTML5 elements and syntax with progressively enhanced components.

Browser support

Bootstrap is tested and supported in major modern browsers like Chrome, Firefox, and Internet Explorer.

  • Latest Chrome
  • Latest Firefox
  • Latest Safari
  • Latest Edge
  • Internet Explorer 10/11
  • Latest Opera

Features & includes

Bootstrap provides an unparalleled array of features and reusable components that let you hit the ground running:

  • Responsive 12-column grid
  • 12 custom, modular jQuery plugins
  • CSS styles for forms, navigation & more
  • Dozens of reusable components
  • Components are scaled according to a range of resolutions and devices
  • Built with Sass (CSS included)
  • Complete styleguide documentation
