DREW is a robust landing page template which aims to solve all your landing page needs. Suits for any business

DREW is made for any business field, means you can use it for any purpose. All the components are built in a general concept to adapt every needs from a marketing landing page. You can use DREW for:

  • Product release page (eBook, App, Software, etc)
  • Event landing page (you can use coming soon countdown / "join event" form)
  • Restaurant landing page (you can use fullscreen images / slider, google maps)
  • Hotel reservation page (put a booking form, google maps)
  • Photography / Creative / Studio / Agency (put a nice portfolio)
  • and many more, your creativity is the limit.

Bootstrap Framework (v3.3) — Bootstrap is one of the most powerful framework, used by thousand sites nowadays. The framework provides great flexibility on the grid system and all UI components.

470++ Icon Integration — Integrated with FontAwesome out of the box. You can use the icons everywhere you want.

Parallax Effect (optional) — Bring 3D effect into your landing page. However, you may want to disable this effect in favor of "better performance".

Modular CSS Classes — Let you easily customize the page with a single class change: Changing section's color scheme, background overlay, background image transparency, etc.

Optimized for Performance — The core files are only less than 1MB! Want to be even faster? I have included some page optimizations tips & tricks for you to follow in the documentation.

Asynchronous Video Embed — Embed Youtube or Vimeo Video asynchronously for the best page load performance.

Subscription Form via Mailchimp Integration — Easily Build your mail list with Mailchimp module, directly configure the backend settings.

Google Maps — Put your business addresses with integrated Google Maps, so everyone can reach you out.

Detailed & Straight Forward Documentation — The documentation was clear and straight forward on step by step to customize all components.

About Bootstrap

Bootstrap is designed to help people of all skill levels – designer or developer, huge nerd or early beginner. Use it as a complete kit or use it to start something more complex.

Unlike other front-end toolkits, Bootstrap was designed first and foremost as a styleguide to document not only its features, but best practices and living, coded examples.

Built to support new HTML5 elements and syntax with progressively enhanced components.

Browser support

Bootstrap is tested and supported in major modern browsers like Chrome, Firefox, and Internet Explorer.

  • Latest Chrome
  • Latest Firefox
  • Latest Safari
  • Latest Edge
  • Internet Explorer 10/11
  • Latest Opera

Features & includes

Bootstrap provides an unparalleled array of features and reusable components that let you hit the ground running:

  • Responsive 12-column grid
  • 12 custom, modular jQuery plugins
  • CSS styles for forms, navigation & more
  • Dozens of reusable components
  • Components are scaled according to a range of resolutions and devices
  • Built with Sass (CSS included)
  • Complete styleguide documentation
