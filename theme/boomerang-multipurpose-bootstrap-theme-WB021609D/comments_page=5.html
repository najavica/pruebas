John1 PURCHASED
5 months ago
I see you use the z-depth-x class to create a box shadow. Is there a way to change the defualt shadow color? I would like to change the color to green (or some other color).

John
webpixels SELLER
5 months ago
Hi, John,

There are 2 ways to do that:

The easy way is to copy the block code coresponsing to .z-depth-* classes and paste it in the custom-style.css from the css folder. After that, change the RGBA values corresponding to the needed color and include this file after the global-style.css in the HEAD section. This should override the default values with the new ones.

The more advanced way is to it with Sass. All you need is to open the _helper.scss file from /assets/sass/core copy the block code coresponsing to .z-depth-* classes and paste it in the _custom.scss files from the same folder. After that, change the RGBA values corresponding to the needed color and compile it to a new css. This will help you have a single and unified stylesheet. The only inconvenient would be that you will have to re-compile every time you want update to a newer version of Boomerang.

I wouldn't recommend to change anything from the global-style.css or the sass files. Use the _custom.scss and custom-style.css to create your own styles. This way you will make sure you can use the upcoming updates without problems.

Let me know if you need further help.

Best regards,
Webpixels Team

joeR57 PURCHASED
5 months ago
Thanks for the great template, I'm having an issue with the slider used on the medical home page. When the screen is less than 992px the height of the slider reduces to around 300px. Is there a way of changing this height. I have looked through the CSS and can't seem to find it.
webpixels SELLER
5 months ago
Hi,

Thank you very much!

We made the slider that way so it can look good on any device. It is a very lightweight script that does that so we can keep the page simple as possible. Sometimes using a premium slider requires too much resources.

But there would be two options:

1. You could create a slider using one of the templates included in the Revolution Slider plugin. There you have much more options regarding the size of the slider.

2. You could modify the lines in the main JS and main CSS for the Boomerang Slider that you are using:

Remove the height: auto from:

@media(max-width: 991px) {
.background-image-holder .holder-item {
height: auto;
}
}

In the main JS folder open wpx.app.js (if your version is below 3.3) or boomerang.min.js (for the 3.3 and above):

Replace the:

$('.background-image-holder').css({
height': 'auto'
});

WITH:

$('.background-image-holder').css({
'height': holderHeight + 'px'
});

This will keep the slider in fullscreen no matter the device's screen size.

If you need further help, email us at [email protected]

Best regards,
Webpixels Team

About Bootstrap

Bootstrap is designed to help people of all skill levels – designer or developer, huge nerd or early beginner. Use it as a complete kit or use it to start something more complex.

Unlike other front-end toolkits, Bootstrap was designed first and foremost as a styleguide to document not only its features, but best practices and living, coded examples.

Built to support new HTML5 elements and syntax with progressively enhanced components.

Browser support

Bootstrap is tested and supported in major modern browsers like Chrome, Firefox, and Internet Explorer.

  • Latest Chrome
  • Latest Firefox
  • Latest Safari
  • Latest Edge
  • Internet Explorer 10/11
  • Latest Opera

Features & includes

Bootstrap provides an unparalleled array of features and reusable components that let you hit the ground running:

  • Responsive 12-column grid
  • 12 custom, modular jQuery plugins
  • CSS styles for forms, navigation & more
  • Dozens of reusable components
  • Components are scaled according to a range of resolutions and devices
  • Built with Sass (CSS included)
  • Complete styleguide documentation
