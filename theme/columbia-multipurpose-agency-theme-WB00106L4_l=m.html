Columbia - Multipurpose Agency Theme

ThemesCollective proudly presents their debut WrapBootstrap theme, 'Columbia'. Designed and built with a 'user first' approach, Columbia offers a beautifully elegant, simple and well-spaced theme that follows the principles of design.

The theme comes packed with many plug 'n' play page elements, that you can use to create endless template layouts, or you can use the ready-built templates that come shipped.

FEATURES:

  • Built with the Bootstrap framework
  • Four Hero variations/styles
    • Static Image
    • Image with colour overlay
    • Slider feature
    • Video Background
  • Multiple block easy-use page elements
  • 2 x Creative pop-up menu's in grid format, for main navigation and social links
  • Scroll-to bullet navigation
  • 2 & 3 column portfolios, plus filter/sortable option
  • Dedicated portfolio item page with carousel/slider and project details
  • Simple blogging solution
  • Dedicated company profile page with team member avatars and company timeline
  • Responsive videos for Youtube, Vimeo etc
  • Smooth page animations
  • Styled and read-to-go Twitter Bootstrap framework
  • Fully working PHP Contact Form with AJAX validation and featured Google Map section
  • Custom scrollbars
  • Fully responsive across desktop, tablet & mobile devices
  • Internet Explorer 8 compatible
  • Fully validated HTML, CSS & JS
  • Fully supported by ThemesCollective customer support 24/7
  • Fully documented with html & PDF versions

About Bootstrap

Bootstrap is designed to help people of all skill levels – designer or developer, huge nerd or early beginner. Use it as a complete kit or use it to start something more complex.

Unlike other front-end toolkits, Bootstrap was designed first and foremost as a styleguide to document not only its features, but best practices and living, coded examples.

Built to support new HTML5 elements and syntax with progressively enhanced components.

Browser support

Bootstrap is tested and supported in major modern browsers like Chrome, Firefox, and Internet Explorer.

  • Latest Chrome
  • Latest Firefox
  • Latest Safari
  • Latest Edge
  • Internet Explorer 10/11
  • Latest Opera

Features & includes

Bootstrap provides an unparalleled array of features and reusable components that let you hit the ground running:

  • Responsive 12-column grid
  • 12 custom, modular jQuery plugins
  • CSS styles for forms, navigation & more
  • Dozens of reusable components
  • Components are scaled according to a range of resolutions and devices
  • Built with Sass (CSS included)
  • Complete styleguide documentation
